🔒 プライバシーポリシー

収集する情報について

本サイト（AUXEN FX Portal）は、GitHub Pages にてホスティングされる静的サイトです。 利用状況の把握と改善のため、プライバシーに配慮したアクセス解析ツール GoatCounter を使用しています。 GoatCounter は個人を特定するための Cookie を使用せず、閲覧者を追跡するための個人情報を保存しません。

GitHub によるアクセスログ

本サイトは GitHub Pages（GitHub, Inc.）のサーバーを利用しています。 GitHub はサーバーのアクセスログ（IPアドレス・アクセス日時等）を収集する場合があります。 これらのデータの取り扱いは GitHub のプライバシーポリシー に従います。

お問い合わせフォームについて

お問い合わせフォーム（Google フォーム）をご利用の場合、入力された情報（氏名・メールアドレス・ お問い合わせ内容）は Google LLC のサーバーに保存され、当サイト運営者が受信します。 取得した情報は、お問い合わせへの回答のみに使用し、第三者へ提供することはありません。

アクセス解析ツールについて

当サイトでは GoatCounter により、ページ閲覧数、参照元、閲覧環境などの統計情報を取得する場合があります。 これらの情報はサイト改善のために利用し、個人を特定する目的では使用しません。

Cookie について

現時点では、当サイト独自の Cookie は使用していません。 ただし、将来の機能追加により Cookie を使用する場合は本ポリシーを更新します。

プライバシーポリシーの変更

本ポリシーは予告なく変更する場合があります。重要な変更がある場合は本ページに掲載します。

お問い合わせ

プライバシーに関するお問い合わせは お問い合わせページ よりご連絡ください。
